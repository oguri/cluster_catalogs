Ref: [M. Oguri, et al., PASJ, 70, S20 (2018)](https://ui.adsabs.harvard.edu/abs/2018PASJ...70S..20O/abstract) 

Ref: [M. Oguri, MNRAS, 444, 147-161 (2014)](https://ui.adsabs.harvard.edu/abs/2014MNRAS.444..147O)
