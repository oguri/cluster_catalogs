# HSC S23B CAMIRA cluster catalogs

CAMIRA HSC final year cluster catalogs presented in Oguri et al. (2026).

Ref: [M. Oguri, et al., PASJ, 78, 416 (2026)](https://ui.adsabs.harvard.edu/abs/2026PASJ...78..416O) 

Ref: [M. Oguri, MNRAS, 444, 147-161 (2014)](https://ui.adsabs.harvard.edu/abs/2014MNRAS.444..147O)

Photometric LRG catalogs may be available [here](https://drive.google.com/drive/folders/19dKiNs7Wdq44X3AtYy2opOf8ulHTLWeM?usp=sharing).

| file       | description |
|:---        |:---     |
| camira_hsc_s23b_wide.tbl | Wide cluster catalogue w/o star mask  |
| camira_hsc_s23b_wide_sm.tbl | Wide cluster catalogue w/ star mask  |
| camira_hsc_s23b_dud.tbl | Deep/UD cluster catalogue w/o star mask |
| camira_hsc_s23b_dud_sm.tbl | Deep/UD cluster catalogue w/ star mask |
